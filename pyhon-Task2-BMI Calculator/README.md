BMI Calculator

📖 Description
A command-line application that calculates a user's Body Mass Index (BMI) based on their weight and height. The program validates user input, calculates the BMI, categorizes it according to standard BMI ranges, and allows users to perform multiple calculations without restarting the program.

✨ Features
Calculates BMI using the standard formula.
Displays BMI rounded to two decimal places.
Categorizes BMI as:
Underweight
Normal
Overweight
Obese
Validates numeric input.
Prevents negative and zero values.
Allows repeated BMI calculations.
Handles invalid input using exception handling.

🛠️ Concepts Used
Functions
While loops
Nested loops
Conditional statements (if, elif, else)
Exception handling (try / except)
Input validation
Return values
Formatted string output (f-strings)
