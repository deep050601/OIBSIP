Password Generator

📖 Description
A Python application that generates secure random passwords based on user preferences. Users can specify the password length and choose whether to include uppercase letters, lowercase letters, numbers, and special characters.

✨ Features
Generates random passwords.
Custom password length.
Optional inclusion of:
Uppercase letters
Lowercase letters
Numbers
Special characters
Validates user input.
Supports generating multiple passwords in one session.
Uses Python's random module for password generation.
