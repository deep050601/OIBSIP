# Random Password Generator

A Python command-line application that generates secure random passwords based on user preferences. Users can choose the password length and select whether to include uppercase letters, lowercase letters, numbers, and special characters.

## ✨ Features

* Generates strong random passwords.
* User-defined password length.
* Optional inclusion of:

  * Uppercase letters
  * Lowercase letters
  * Numbers
  * Special characters
* Validates user input.
* Handles invalid entries using exception handling.
* Allows users to generate multiple passwords in one session.

## 🛠️ Technologies Used

* Python 3
* Built-in `random` module
* Built-in `string` module (if used)

## 📚 Concepts Practiced

* Functions
* Loops
* Lists
* Randomization
* String manipulation
* Conditional statements
* Exception handling
* Input validation

## 🚀 How to Run

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:

```bash
python random_password_generator.py
```

## 🔒 Why This Project?

Strong passwords are essential for protecting online accounts. This project demonstrates how Python can be used to generate customizable and secure random passwords while practicing fundamental programming concepts.
