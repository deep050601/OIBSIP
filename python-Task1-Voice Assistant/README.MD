## 🎙️ Voice Assistant using Python

A simple voice-controlled assistant built with Python that can listen to your voice, respond using text-to-speech, tell the current time and date, perform Google searches, and open popular websites.

---

## 📌 Features

- 🎤 Capture voice input using your microphone
- 🗣️ Respond to greetings like **"Hello"**
- 🕒 Tell the current time
- 📅 Tell the current date
- 🔍 Search Google using voice commands
- 🌐 Open Google and YouTube
- 🔊 Respond using offline Text-to-Speech (`pyttsx3`)
- ⚠️ Gracefully handle speech recognition errors
- 👋 Exit the assistant using commands like **"Goodbye"**

---

## 🛠️ Technologies Used

- SpeechRecognition

## 🎤 Supported Voice Commands

| Command | Action |
|---------|--------|
| Hello | Assistant greets you |
| Time | Tells the current time |
| Date | Tells today's date |
| Search | Searches Google for your query |
| Open Google | Opens Google |
| Open YouTube | Opens YouTube |
| Goodbye / Bye | Closes the assistant |

---

## 📸 Sample Interaction

```
Assistant: Good Morning!
Assistant: Hello! How may I assist you?

You: What is the time?

Assistant: The current time is 10:30 AM.

You: Search

Assistant: What do you want to search for?

You: Python tutorials

Assistant: Searching for Python tutorials...
```
