# 💬 Python Chat Application

A simple real-time chat application built using **Python**, **Socket Programming**, and **Multithreading**. The application allows multiple users to communicate over a local network (`localhost`) with timestamped messages and graceful disconnection handling.

---

## 🚀 Features

- Real-time communication between connected clients
- Multi-client support using threads
- Timestamped messages
- Graceful client disconnection handling
- Server broadcasts messages to all connected clients
- Runs locally using `localhost`

---

## 🛠️ Technologies Used

- Python 3
- Socket Programming
- Multithreading
- Datetime Module

## ▶️ How to Run

### 1. Start the Server

```bash
python server.py
```

### 2. Start the Client

Open another terminal and run:

```bash
python client.py
```

Run multiple client instances to simulate multiple users chatting.
